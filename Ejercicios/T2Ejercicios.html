Sean $x,y,z \in \mathbb{R}$. Establezca cual o cuales axiomas de los números reales justifican cada una de las siguientes igualdades:

  1. $(6+8)x=x(6+8)$
  2. $(x+3)y + 2= (x\cdot y + 3 \cdot y) + 2$
  3. $(3 + 5) + 2 = 3 + (5 + 2)$
  4. $(2 + 3) + 5 = 2 + (3 + 5)$
  5. $[(1)(2)](3) = [(2)(1)](3)$
  6. $(x + y) + 3 = (y + x) + 3$
  7. $(1 + 2)(-3) = 1(-3) + 2(-3)$
  8. $[(w + 3)2]z = [2(w + 3)]z$
  9. $(-13 + z)(2) + 7 = [z + (-13)](2) + 7$
  10. $(a - b)+ [-(a - b)] = 0$
  11. $(3 + 4)(5 + 2) = (3 + 4)5 + (3 + 4)2$
  12. $x(y + 0) + z = xy + z$
  13. $(x + 2) + [-(x + 2)] = 0$
  14. $2(x+y)= 2x + 2y$
  15. $(x+5)+y= y + (x+5)$
  16. $2(3z)= (2 \cdot 3) z$
  17. $y + (x+y)= (y + x) + y$
  18. $5(4 + 7)= 5 (7 + 4)$
  19. $(8 + a)b = 8b + a b$
  20. $(-1)[-3 + 4 ] = (-1)[-3] + (-1)[4] $
